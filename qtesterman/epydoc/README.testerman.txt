Introduction
~~~~~~~~~~~~
    Epydoc is a tool for generating API documentation for Python
    modules, based on their docstrings.  A lightweight markup language
    called epytext can be used to format docstrings, and to add
    information about specific fields, such as parameters and instance
    variables.

Modifications for Testerman integration
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    This source tree contains only a part of the complete epydoc
    distribution. In particular, unit tests, gui support, scripts and
		epydoc documentation have not been included.
		It should not be considered as a valid, redistributable epydoc
		packaging for non-qtesterman usage.

    The included code, however, has been left unmodified and is
    from epydoc 3.0.1.

    If you need epydoc, please go to its homepage on:
        <http://epydoc.sourceforge.net/>
